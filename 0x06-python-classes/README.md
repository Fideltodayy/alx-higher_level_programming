# 0x06 Python - Classes and Objects

This project covers the concept of classes and objects in Python. In object-oriented programming, a class is a blueprint for creating objects (a particular data structure), providing initial values for state (attributes or properties), and implementations of behavior (methods or functions).

## Learning Objectives

By the end of this project, you should be able to:

- Understand the basics of object-oriented programming (OOP)
- Define and create classes and objects in Python
- Use attributes and methods to manipulate objects
- Understand the difference between class attributes and instance attributes
- Implement inheritance and understand the concept of superclasses and subclasses
- Use encapsulation, abstraction, and polymorphism in Python

## Files

The following files are included in this project:

- `0-square.py`: An empty class `Square` that defines a square.
- `1-square.py`: A class `Square` that defines a square with a private instance attribute `size`.
- `2-square.py`: A class `Square` that defines a square with a private instance attribute `size` and a public instance method `area`.
- `3-square.py`: A class `Square` that defines a square with a private instance attribute `size` and public instance methods `area` and `my_print`.
- `4-square.py`: A class `Square` that defines a square with a private instance attribute `size` and public instance methods `area` and `my_print`. It also includes getter and setter methods for the `size` attribute.
- `5-square.py`: A class `Square` that defines a square with a private instance attribute `size` and public instance methods `area` and `my_print`. It also includes getter and setter methods for the `size` attribute, and a class method `def bigger_or_equal(cls, rect_1, rect_2):` that returns the biggest rectangle based on the area.
- `6-square.py`: A class `Square` that defines a square with a private instance attribute `size` and public instance methods `area` and `my_print`. It also includes getter and setter methods for the `size` attribute, and a class method `def bigger_or_equal(cls, rect_1, rect_2):` that returns the biggest rectangle based on the area. Additionally, it includes a static method `def square_area(size):` that calculates the area of a square.
